The Frameworks

Distributed Actors a la Akka

https://github.com/ajlopez/SimpleActors
  • Inspired by Akka (Java/Scala)
  • Each node has an actor tree
  • Each actor can be references by address (host@name/name)
  • An actor can create other actors
## System and Actors ```js // System var webcrawler = node.create('webcrawler'); // Actors var downloaderref = webcrawler.actorOf(downloader, 'downloader', { router: true }); var resolverref = webcrawler.actorOf(resolver, 'resolver'); var harvesterref = webcrawler.actorOf(harvester, 'harvester'); var serverref = webcrawler.actorOf(server, 'server'); ```
